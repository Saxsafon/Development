### Редактирование кода

```console
ctrl + alt + T - вставить окружение
ctrl + d - duplicate line and caret 
ctrl + y - delete the whole line

ctrl + w - select word at caret
    + ctrl + w - select whole object
    + ctrl + shift + w - return to previous selection

alt + j - multiselect word
    
ctrl + z - undo the change
ctrl + shift + z - rollback changes
ctrl + shift + r - replace in files

ctrl + / - comment the lines
ctrl + shift + j - join lines

F2 or shift + F2 - next error
alt + enter - fix error

shift + F6 - rename variable
shift + F10 - run
```

### Исследование кода

```console
page down - go down
page up - go up
home - to the begining of the line
end - to the end of the line

ctrl + p - show parameters of the method call at the caret
ctrl + q - see documentation of the method or function

ctrl + "-" - collapse all
ctrl + "+" - expand all
```

### Git

```console
ctrl + k or alt + 0 - open commit panel
alt + 9 - open Git panel
```

### Навигация

```console
alt + left - navigate to the left opened file
alt + right - navigate to the right opened file
ctrl + alt + left - navigate back
ctrl + alt + right - navigate forward

alt + 0 - open commit panel
alt + 1 - show project bar
alt + 2 - open favorites panel
alt + 4 - open run panel
alt + 5 - open debug panel
alt + 6 - open problems panel
alt + 7 - open structure panel
alt + 8 - open services panel
alt + F12 - open Terminal panel
alt + home - open navigation bar

ctrl + alt + S - settings

ctrl + e - show the list of recently opened files
    + ctrl + e - show edited only

ctrl + F4 - close current file
```

### Поиск

```console
ctrl + F
ctrl + shift + F
ctrl + shift + A - find action
double shift - search everywhere

alt + 3 - open find panel
alt + F7 - find usages of the element
```
